# Install Jest:

    yarn add --dev jest

    Or npm:

    npm install --save-dev jest

Add the following section to your package.json:

    {
        "scripts": {
            "test": "jest"
        }
    }

## run test

    npm run test

## Run a specific test file

    npm test -- SomeTestFileToRun
